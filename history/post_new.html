YearOccurence
1847 George Boole published a paper on symbolic logics called The Mathematical Analysis of Logic
1879 Gottlob Frege produced a logic that, except for some notational changes, forms the first-order logic that is used today as the most basic knowledge representation system
1900 David Hilbert presented a list of 23 problems that he correctly predicted would occupy mathematicians for the bulk of century
1930 Kurt Godel showed that there exist an effective procedure to prove any true statement in the first-order logic of Frege-Russell
1931 Kurt Godel showed that real limits do exist. His incompleteness theorem showed that in any language expressive to describe the properties of the natural numbers, there are true statements that are undecidable: their truth cannot be established by any algorithm
1940 Alan Turing's team built the first operational modern computer "Heath Robinson" to decipher the German messages
1941 Konard Zuse invented Z-3, the first operational programmable computer. Zuse invented floating point number for Z-3
1942 John Atanasoff and his graduate student Clifford Berry assembled the first electronic computer, the ABC
1943 McCullouch and Pitts designed Turing-complete "artificial neurons"
1945 Konard Zuse developed Plankalkul, the first high level programming language
1952 Nathaniel Rochester and his group build IBM 701, this was the first computer to yield income for its manufacturer
1956
  • The name artificial intelligence was coined by John McCarthy at Dartmouth College
  • Newell and Simon demonstrated their reasoning program called Logic Theorist (LT)
  • In February, Arthur Samuel demonstrated the program for playing checkers which could learn new moves on its own

    1958 John McCarthy published a paper entitled Programs with Common Sense, in which he described a hypothetical program called 'Advice Taker', which can be seen as first complete AI system
    1959 Herbert Gelernter constructed the Geometry Theorem Prover
    1961 Newell and Simon developed GPS (General Problem Solver)
    1963 James Slagle's SAINT program was able to solve closed-form integration problems typical of first year college calculus courses
    1968
  • Tom Evan's ANALOGY program solved geometric analogy problems that appear in IQ test
  • Bertram Raphael's SIR (Semantic Information Retrieval) was able to accept input statements in a very restricted subset of English and answer question thereon
    		•
    1997 Deep Blue became the first computer chess-playing system to beat a reigning world class champion, Garry Kasparov
    2011 Watson, IBM's question answering system, in a quiz show called Jeopardy defeated the two greatest Jeopardy champions, Brad Rutter and Ken Jennings
    2016 AlphaGo won 4 out of 5 games of Go in a match with Go champion Lee Sedol, becoming the first computer Go-playing system to beat a professional Go player without handicaps
